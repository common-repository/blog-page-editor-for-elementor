== Description ==

Blog Page Editor For Elementor allows you to edit the blog page content with Elementor.

Usually Elementor doesn't allow you to edit the blog page content, with Blog Page Editor For Elementor you can do it.

However, it depends on your theme if the blog page content appears on the blog page. Blog Page Editor For Elementor works only in the backend with the Elementor editor. What happens on the frontend is not under the control of this plugin, but under the control of your theme.


== How to edit the content of the blog page with Elementor  ==
Install Blog Page Editor For Elementor, and that's it. No settings required for this plugin. After activation you will be able to edit the blog page with the editor of Elementor.
